# Telecom Customer Churn Analysis Dashboard

## Project Overview

This project analyzes customer churn behavior in a telecom company using Power BI.

The objective is to identify key churn drivers, customer risk segments, and actionable business insights that can improve customer retention.

---

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel
- Data Visualization

---

## Key Metrics

- Total Customers: 7,043
- Churned Customers: 1,869
- Active Customers: 5,174
- Churn Rate: 26.54%

---

## Dashboard Pages

### 1. Executive Churn Overview

Provides a high-level view of:

- Customer churn rate
- Contract-based churn analysis
- Tenure-based churn analysis
- Internet service churn distribution
- Executive business insights

### 2. Customer Risk Analysis

Provides deeper analysis of:

- Payment method impact on churn
- Customer segment analysis
- Gender distribution
- Billing preference behavior
- Monthly charge analysis

---

## Key Findings

- Month-to-month contracts exhibit the highest churn levels.
- Fiber optic customers account for the majority of churned customers.
- Electronic Check users show significantly higher churn behavior.
- Customers with higher monthly charges are more likely to churn.
- Gender has minimal impact on customer churn.
